<!-- Wordly Dictionary SPA — Summary

Wordly Dictionary SPA is a single-page web application that allows users to search for English words and instantly view their meanings, pronunciation, synonyms, and related information without refreshing the page.

The application uses the Free Dictionary API to fetch real-time word data and displays it dynamically using JavaScript DOM manipulation.

Key Features:
Word search functionality
Displays definitions and parts of speech
Shows pronunciation (text and audio when available)
Displays synonyms when available
Handles errors like invalid or empty input
Dynamic updates without page reload
Responsive design using Bootstrap

Technologies Used:
HTML for structure
CSS for custom styling
Bootstrap for responsive UI
JavaScript for logic and interactivity
Fetch API for retrieving data

How It Works:
The User enters a word
JavaScript captures input via event listener
It then makes request through Fetch event which sent to the dictionary API
The API returns word data
Data is parsed and displayed on the page dynamically
Errors are handled gracefully with messages

Outcome:

The project demonstrates how to build a functional SPA using APIs, event handling, and DOM manipulation while maintaining a clean and responsive user interface. -->
